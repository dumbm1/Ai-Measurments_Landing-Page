Установите

Автоустановка

  • УстановитеZXPInstallerпо завершению процесса установки снимите флаг "Запустить ZXPInstaller" и закройте окно установщика
  • Теперь запустите ZXPInstaller
  • Перетащите файл ai_measurments_1.0.0.zxp в окно ZXPInstaller
  • Перезагрузите Иллюстратор, пользуйтесь!

Для продвинутых и бесстрашных - ручная установка!

  • Переименуйте ai_measurments_1.0.0.zxp в ai_measurments_1.0.0.zip
  • Распакуйте zip-archive в папку ai_measurments_1.0.0
  • Переместите папку ai_measurments_1.0.0 по одному из следующих путей:
    • для всех пользователей:
      • win(x64): C:\Program Files (x86)\Common Files\Adobe\CEP\extensions, или C:\Program Files\Common Files\Adobe\CEP\extensions
      • macOS: /Library/Application Support/Adobe/CEP/extensions
    • для одного пользователя:
      • win: C:\Users\<ИМЯ ПОЛЬЗОВАТЕЛЯ>\AppData\ Roaming\Adobe\CEP\extensions
      • macOS: ~/Library/Application Support/Adobe/CEP/extensions

    • Папка 'CEP\extensions' не существует? Чтож, тогда просто создаем ее вручную.

Пользуйтесь

Как проставить линейные размеры:

  • Выделите один или несколько объектов
  • Нажмите одну из кнопок на панели: L (left), R (right), T (top), B (buttom)

Радиус и диаметр:

  • Выделите окружность
  • Нажмите кнопку r (radius) или D (diameter)

Как проставить размер между двумя объектами?

  • Выделите два объекта
  • Hold down the Crtl/Cmd key and click the some size button
  • Нажмите и удерживайте клавишу Crtl/Cmd
  • Проставьте нужный линейный размер (кнопки L,R,T,B)
  • Дополнительно: все еще удерживая клавишу Crtl/Cmd измените параметры разметки
