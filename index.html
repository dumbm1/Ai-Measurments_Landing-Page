Install & go!

It's easy - autoinstall!

  • InstallZXPInstaller, and run it
  • Drag'n'Drop the ai_measurments_1.0.0.zxp to ZXPInstaller window
  • Refrash the Illustrator, enjoy!

For advenced and courage - manual installation!

  • Rename ai_measurments_1.0.0.zxp to ai_measurments_1.0.0.zip
  • Unpack zip-archive to folder ai_measurments_1.0.0
  • Drag'n'Drop the folder ai_measurments_1.0.0 to one of that paths:
    • for all users:
      • win(x64): C:\Program Files (x86)\Common Files\Adobe\CEP\extensions, or C:\Program Files\Common Files\Adobe\CEP\extensions
      • macOS: /Library/Application Support/Adobe/CEP/extensions
    • for one user:
      • win: C:\Users\USERNAME\AppData\ Roaming\Adobe\CEP\extensions
      • macOS: ~/Library/Application Support/Adobe/CEP/extensions

    • Path 'CEP\extensions' does not exists? You must make that folders manualy.

Enjoy this- save you time & brain cells!

Linear measurments lines:

  • Select you object that need to measure
  • Push the one of that buttons L (left), R (right), T (top), B (buttom)

Radius & Diameter:

  • Select a circle which you want to measure.
  • Push the one of that buttons r (radius), D (diameter)

How make markup dimensions between two objects?

  • Select two targets
  • Hold down the Crtl/Cmd key and click the some size button
  • Optionally: while still holding Crtl/Cmd adjust the markup options
